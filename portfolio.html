TO PLAY OUR GAME CLICK HERE!

Our game is called Anime Clicker. The concept of the game is to click
on the icon to accumulate points to move on the next level. When clicked
the points will add up randomly from 7 to 10. The game gets more difficult
as you move up levels since the ammount of time decreases and the points you
need to accumulate increases. The user can lose when they don't accumulate
enough points in a certain amount of time to move on the next level the stage
will reset and you'll have to start over again.

Our inspiration to make this game was cookie clicker as there's a point
system and it seemed easy to make a different version of the game. Two
successful points that we had for the coding of the game was the reset system
and the point system of the game. Two obstacles that we had in the development
of our game was the timer and points system because at first when we made it
the points and timer would work for all of the sprites since we didn't know
that you can make a variable for one sprite only. If we had more time for the
game we would've made the game look more aesthetically pleasing because the
layout for our game was very basic.

This algorithm contributes a lot to our game since it was a way for the
user to lose and made it more difficult to complete the stages of the game. This is also included in our more blocks system which was a reset system
to reset the points and timer to 0 when it reached a certain time frame.
